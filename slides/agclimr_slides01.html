Agroclimate Metrics

Agroclimate metrics are proxies of abiotic factors that influence the development of plants and insects.

Examples:

  • degree days

  • first spring frost

R-tricks: Piping

Piping syntax is an alternative way of chaining functions together than nested parentheses:

zoo(moo(boo(foo(99),k=7),n=4))

With piping, you use the pipe operator |> or %>% to ‘feed’ the result of one function as the first argument to the next function.

  • Keyboard shortcut for inserting the pipe operator: ctrl + shift + m
    You can tell RStudio which pipe to insert under Tools >> Global Options >> Code
    |> (‘native’ pipe introduced R4.0)
    %>% (traditional, requires magrittr package)

R-tricks: Dates

Agroclimate metrics frequently requires working with date values.

These are not Date objects:

x <- "2021-06-15"

y <- "4/1/2019"


To convert date-formatted text to Date objects use as.Date():

bloom_date <- as.Date("2021-04-15")
class(bloom_date)
## [1] "Date"
